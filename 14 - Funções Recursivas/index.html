Criando Funçãoes Recursivas

O que é uma função recursiva

Uma função recursiva é uma função que chama ela mesma para resolver um problema menor.
Por exemplo, uma função que calcula a soma de todos os números de 1 a N pode ser implementada como:
function somaRecursiva(n) { if (n === 1) { return 1; } else { return n + somaRecursiva(n - 1); } }
